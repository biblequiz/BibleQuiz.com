## What's the Point?

The BibleQuiz.com wiki allows the community to contribute to a knowledge base for the entire Bible Quiz community. The exact organization is a work in progress. The goal is to capture instructions on how to use the apps, how to run meets, how to get started as a team, etc.

## How does it work?

> [!TIP]
> The `Edit page` link won't appear on the mobile UI.

1. If you want to edit a page, click the `Edit page` link at the top of any wiki page. To create a new page, click `Edit page` link on any page and then click `New page`.
2. You will need a GitHub account to contribute to pages.
3. Any change you will make will appear in the [GitHub Wiki](https://github.com/biblequiz/BibleQuiz.com/wiki).
> [!NOTE]
> Changes are synced from [GitHub Wiki](https://github.com/biblequiz/BibleQuiz.com/wiki) a few times a day. This may result in cases where you observe a mismatch between the [GitHub Wiki](https://github.com/biblequiz/BibleQuiz.com/wiki) and BibleQuiz.com.
